By Sepand Haghighi

 

Free Project For Creating Academic Home page

** This version of QPage has 4 default pages (Home , Project , Resume , About) **

 

Please Follow This Steps

 

(1) Download And Install Last Version Of Python (1)

(2) Download QPage (2)

(3) Put your profile picture in image folder (3)

(4) Put your each page detail in a text file with page name in doc folder (* In this step you can use [M] , [S] or [L] at the end of each line to control font size ) (4)

(5) Put your resume file (pdf format) in doc folder (5)

(6) Run setup.py (6)

(7) Choose your text and background color (7)

(8) Finished!!! Now you can go and upload output folder contains directly in your host (8)
