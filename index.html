Quick Guide

 

(1) Download And Install Last Version Of Python3
( * or use exe-file and skip this step * )

(2) Download (Source) or Download (Exe-File) of QPage

(3) Put your profile picture in image folder

(4) Put details of each of your pages in a text file with page name in doc folder

(5) Put your resume file (pdf format) in doc folder

(6) Put your font file in font folder (*Optional)

(7) Run setup.py

(8) Choose your profile image size

(9) Choose your text and background color

(10) Finished!!! Now you can go and upload output folder's content directly in your host
